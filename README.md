# 🌸 Iris Flower Classification & Deployment 🌸

Welcome to my first End-to-End Machine Learning project! I built a model to identify Iris species and deployed it as an interactive web application.

## 🚀 Live Demo
You can try out the live web app here: 
👉 **[Click Here to Predict Iris Flowers]https://iris-flower-kaveesha.streamlit.app/**

## ✨ Project Overview
This project covers the entire ML pipeline:
*   **Data Exploration:** Understanding the structure of the Iris dataset.
*   **Visualization:** Creating insightful charts to see patterns in nature.
*   **Modeling:** Building a Support Vector Machine (SVM) model for classification.
*   **Deployment:** Creating a user-friendly web interface using Streamlit.

## 🛠️ Tools & Technologies
*   **Python:** Core programming language.
*   **Scikit-Learn:** For building and saving the SVM model.
*   **Pandas & NumPy:** For data manipulation.
*   **Seaborn & Matplotlib:** For data visualization.
*   **Streamlit:** For the web application interface.
*   **GitHub:** For version control and hosting.

## 🖥️ Web App Preview
The app allows users to input flower measurements (Sepal/Petal length and width) and get an instant prediction of the species along with a corresponding image.

## 📂 How to Run Locally
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the app: `streamlit run app.py`.
